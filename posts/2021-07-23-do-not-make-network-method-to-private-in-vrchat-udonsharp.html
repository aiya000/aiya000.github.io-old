UdonSharpでメソッドをthis.SendCustomNetworkEvent()で呼び出すときは、メソッドをprivateにしない
  • VRChat

❌ 「howdy :)」が表示されない

public void Start() {
  this.SendCustomNetworkEvent(NetworkEventTarget.All, "Foo");
}

private void Foo() {
  Debug.LogError("howdy :)");
}

⭕ 「howdy :)」が表示される

public void Start() {
  this.SendCustomNetworkEvent(NetworkEventTarget.All, "Foo");
}

// Not `private`
public void Foo() {
  Debug.LogError("howdy :)");
}

当たり前だけど！！！
