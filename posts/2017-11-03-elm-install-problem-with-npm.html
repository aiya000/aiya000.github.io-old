elmのインストールがnpmで失敗したからyarnでやったらできた
  • elm

　公式に書いてあるLinuxのインストール方法であるこいつ、なんぞ失敗しよる。

$ sudo npm install -g elm

npm　もういい！もどれ！

いけ、yarn！

$ sudo yarn global add elm

こうすればうまくインストールできた。
