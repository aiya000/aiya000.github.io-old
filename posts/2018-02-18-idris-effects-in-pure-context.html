Run a program of Eff in pure contexts
  • Idris

In a situation, I have a program (Eff) with a STATE effect, and the program is pure.

It can be run in a pure context like below ğŸ˜‚

import Control.Monad.Identity
import Effect.State
import Effects

main : IO ()
main = do
  let Id (_ ** [s]) = runEnv {m=Identity} [default] $ update (+1)
  printLn s
-- {output}
-- 1

PS

I took a Pull Request, Above code can be written as below (no Identity is needed), if the PR is approved ğŸ•

import Effect.State

  printLn s
